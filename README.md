# 🏥 Healthcare Management System

A full-stack **Healthcare Management System** with role-based login for **Patients**, **Doctors**, and **Admins**.

### ✅ Key Features

- 🩺 Appointment booking  
- 👨‍⚕️ Doctor search  
- ⭐ Feedback & rating system  
- 📅 Doctor scheduling  
- ⏰ Reminders  
- 🛡️ Admin moderation
-  AI chatbot with speech to text
